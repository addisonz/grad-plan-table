Network Administrator/Analyst

This program provides the skills that are required to design, manage, troubleshoot, and secure a network environment. Platforms include Windows 7 and 8, Windows Server 2008 and 2012, Linux, Cisco and the Internet. Skill development includes data communications, TCP/IP, hardware, software, network operating systems, and security. I am interested in this program because I want to be able to fix any problems that may develop inside of a network. I want to fix those problems so the network can run smoothly.
